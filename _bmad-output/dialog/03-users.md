# Target Users

## Process
User acknowledged not having deep behavioral knowledge yet (first engagement with client). Saga conducted research to infer typical daily activities of educational administrative staff in Latin American schools. User confirmed the inferred profiles as valid working assumptions.

## Primary User: Administrativo Académico

### Daily Routine (Inferred)
- Morning: Review attendance, process justifications, answer inquiries
- Midday: Enrollment management, section changes, student records
- Afternoon: Reports for management/regulatory entities, pending grades
- Peak stress: End-of-period closing — chasing teachers, reconciling grades, Carmenta digitization

### Frustrations (from Runachay feedback)
- Double entry: SIS + Carmenta
- System bottlenecks: one unclosed section blocks reports
- Parent calls for info the system should show
- Previous implementation ignored admin needs

### Goals
- Speed: process enrollments in minutes
- Reliability: trust the data
- Traceability: audit trail
- Control: dashboard visibility

## Secondary Users
- Docente: Fast attendance/grade entry, no burnout
- Estudiante: Immediate grade visibility, clear navigation
- Padre de Familia: Simple access to child info
- Junta de Accionistas: ROI metrics, regulatory compliance
