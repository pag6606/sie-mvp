# 2.5-ingreso-notas

| Property | Value |
|----------|-------|
| **Scenario** | 02-diana-opera-aula |
| **Page Type** | Full Page |
| **Viewport** | Desktop-first (grilla densa, mejor en monitor) |
| **Visibility** | Docente (solo secciones propias) |

## Purpose
Grilla editable estudiante × componente con cálculo automático del promedio final.

## Layout
```
[← Matemáticas 10-A]              Notas              [Guardar cambios]

┌──────────────────────────────────────────────────────────────┐
│ Estudiante    │ Parcial 1 30% │ Proyecto 40% │ Final 30% │ Final│
│───────────────│────────────────│──────────────│───────────│─────│
│ Andrea López  │    [ 18]      │    [ 16]     │  [ 16]    │ 16.6│
│ Bruno Martínez│    [ 12]      │    [ 14]     │  [  —]    │  — ⚠│
│ Carla Ruiz    │    [ 20]      │    [  —]     │  [ 18]    │  — ⚠│
└──────────────────────────────────────────────────────────────┘

⚠ 2 estudiantes sin nota  ·  Promedio sección: 15.2
```

## Objects

| ID | Component | ES | Behavior |
|----|-----------|----|---------|
| `notas-save` | Button primary | "Guardar cambios" | PATCH /notas/bulk. Confirmación toast |
| `notas-header` | Table header | Nombre componente + peso | Congelado al hacer scroll horizontal |
| `notas-cell` | Input number | — | Escala 0-20. Validación en blur. Tab para siguiente celda |
| `notas-final` | Text (readonly) | — | Cálculo en vivo: Σ(nota × peso/100). Muestra "—" si falta algún componente |
| `notas-warning` | Alert | "X estudiantes sin nota · Promedio sección: X.X" | Siempre visible si hay celdas vacías |
| `notas-log` | Hidden | — | Cada cambio registra: autor, fecha, valor anterior, valor nuevo |

## States
- **Default:** Grilla con notas existentes. Celdas vacías muestran placeholder "—"
- **Sin esquema definido:** Mensaje "Define el esquema de evaluación primero" + link a Esquema
- **Edición:** Celda activa con borde azul. Tab/shift+tab navega entre celdas
- **Nota fuera de rango:** Borde rojo + tooltip "La nota debe estar entre 0 y 20"
- **Esquema congelado:** Si ya se ingresó la primera nota de la sección, los pesos no se pueden modificar (se maneja desde la página de esquema)
- **Save success:** Toast "Notas guardadas" + celdas vuelven a estado readonly visual

## Calculations
```
notaFinal = Σ (notaComponente × pesoComponente / 100)
```
Solo se muestra si TODOS los componentes tienen nota. Redondeo a 1 decimal.

## API
`GET /api/secciones/{id}/notas` → `[{matriculaId, estudiante, notas: [{componenteId, valor}]}]`
`PATCH /api/secciones/{id}/notas` → `[{matriculaId, componenteId, valor}]`
