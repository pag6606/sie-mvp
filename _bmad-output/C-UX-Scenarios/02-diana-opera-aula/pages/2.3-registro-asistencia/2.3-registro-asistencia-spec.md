# 2.3-registro-asistencia

| Property | Value |
|----------|-------|
| **Scenario** | 02-diana-opera-aula |
| **Page Type** | Full Page |
| **Viewport** | Desktop-first, touch-friendly (tablet en aula) |
| **Visibility** | Docente (solo secciones propias) |

## Purpose
Registrar asistencia diaria para toda la sección en segundos con selector rápido por estudiante.

## Layout
```
[← Matemáticas 10-A]    Registro de Asistencia     [02/Jun/2026]

Selección rápida: [Todos presentes] [Todos ausentes]

┌──────────────────────────────────────────────────────────┐
│ Estudiante              Estado              % Asistencia │
│──────────────────────────────────────────────────────────│
│ Andrea López           [✓ Presente ▼]          92% 🟢   │
│ Bruno Martínez         [✗ Ausente ▼]           78% 🟡 ⚠ │
│ Carla Ruiz             [— Justificada ▼]       95% 🟢   │
└──────────────────────────────────────────────────────────┘

⚠ Bruno Martínez: 78% — por debajo del 80% mínimo

[Guardar asistencia]
```

## Objects

| ID | Component | ES | Behavior |
|----|-----------|----|---------|
| `asistencia-date` | DatePicker | Fecha actual | Fecha pasada editable (ventana 7 días). Futura bloqueada |
| `asistencia-all-present` | Button outline | "Todos presentes" | Setea todos a ✓ |
| `asistencia-all-absent` | Button outline | "Todos ausentes" | Setea todos a ✗ |
| `asistencia-row-select` | Select inline | 3 opciones | Presente/Ausente/Justificada. Color verde/rojo/naranja |
| `asistencia-pct` | Badge | "% + 🟢🟡🔴" | Se actualiza en vivo al cambiar selector. Verde ≥80%, Amarillo 70-79%, Rojo <70% |
| `asistencia-warning` | Alert | Texto dinámico | Solo visible si hay estudiantes bajo el mínimo |
| `asistencia-save` | Button primary | "Guardar asistencia" | POST /asistencia/bulk |

## States
- **Default:** Fecha de hoy, todos en blanco si no hay registro previo
- **Con datos previos:** Carga los estados guardados para la fecha seleccionada
- **Fecha futura:** Selectores disabled + mensaje "No puedes registrar asistencia a futuro"
- **Fuera de ventana:** Selectores disabled + "Fuera de la ventana de edición (7 días)"
- **Save success:** Toast verde "Asistencia guardada"

## API
`POST /api/secciones/{id}/asistencia` → `[{matriculaId, fecha, estado: PRESENTE|AUSENTE|JUSTIFICADO}]`
`GET /api/secciones/{id}/asistencia?fecha=YYYY-MM-DD`
