# 1.4-revisar-ajustar

| Property | Value |
|----------|-------|
| **Scenario** | 01-alma-configura-periodo |
| **Page Type** | Full Page |
| **Viewport** | Desktop-first |
| **Visibility** | Admin |

## Purpose
Paso 3 del flujo guiado: revisar secciones clonadas, editar en línea y confirmar.

## Layout
```
[← Volver]        Paso 3 de 4 · 22 de 24 revisadas    [👤]
[████████████████████░░] progress bar

[Buscar...] [Curso ▼] [Docente ▼] [⚠ Pendientes]

┌──────────────────────────────────────────────────────────┐
│ Sección      Curso          Docente       Aula   Estado  │
│──────────────────────────────────────────────────────────│
│ MAT-101-A  Matemáticas 10A  [Diana R. ▼]  [A-12]   ✓   │
│ FIS-101-A  Física 10A       Carlos M.     Lab-3    ✓   │

│ HIS-101-A  Historia 10A     Laura P.      A-08     ✓   │

[24 secciones · 2 pendientes]     [Continuar] (disabled)
```

## Objects

| ID | Component | ES | Behavior |
|----|-----------|----|---------|
| `revisar-search` | Input | "Buscar por curso o sección..." | Filtro en vivo |
| `revisar-filter-curso` | Select | "Curso" | Filtra tabla |
| `revisar-filter-docente` | Select | "Docente" | Filtra tabla |
| `revisar-filter-pendientes` | Button | "Pendientes" | Muestra solo ⚠ |
| `revisar-table` | Table (shadcn) | — | Filas editables inline |
| `revisar-cell-docente` | Select inline | — | Dropdown de docentes activos |
| `revisar-cell-aula` | Input inline | "Aula" | Texto libre |
| `revisar-check` | Checkbox | — | Marca sección como revisada ✓ |
| `revisar-progress` | Progress bar | "X de Y revisadas" | Se actualiza al marcar checks |
| `revisar-cta` | Button primary | "Continuar" | Disabled hasta 100%. PATCH /secciones/bulk |

## States
- **Default:** Tabla con todas las secciones en BORRADOR
- **Pendientes filtrados:** Solo filas con docente sin asignar o aula vacía
- **Completado:** Progress bar 100%, CTA habilitado
- **Empty (empezar desde cero):** "No hay secciones. Crea tu primera sección" + botón "Nueva sección"

## API
`GET /api/secciones?periodo=X` `PATCH /api/secciones/bulk` → `[{id, docenteId, aula, estado}]`
