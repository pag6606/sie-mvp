# 1.2-crear-periodo

| Property | Value |
|----------|-------|
| **Scenario** | 01-alma-configura-periodo |
| **Page Type** | Full Page |
| **Viewport** | Desktop-first |
| **Visibility** | Admin |

## Purpose
Paso 1 del flujo guiado: crear un período académico con 4 campos mínimos.

## Layout
```
[← Volver]              Paso 1 de 4                 [👤]
        ● Crear período  ○ Secciones  ○ Revisar  ○ Confirmar
┌──────────────────────────────────────────────────────┐
│  Código                                              │
│  [_____________]  (prellenado: "2026-2")             │
│                                                      │
│  Nombre                                              │
│  [_____________]  (prellenado: "Período 2026-2")     │
│                                                      │
│  Fecha de inicio        Fecha de fin                 │
│  [01/09/2026]           [15/12/2026]                 │
│                                                      │
│  [Continuar →]                                       │
└──────────────────────────────────────────────────────┘
```

## Objects

| ID | Component | ES | Behavior |
|----|-----------|----|---------|
| `periodo-codigo-input` | Input (shadcn) | "Código" | Validación: único. Ej: "2026-2" |
| `periodo-nombre-input` | Input (shadcn) | "Nombre" | Autocompleta al escribir código |
| `periodo-inicio-input` | DatePicker (shadcn) | "Fecha de inicio" | No anterior a hoy |
| `periodo-fin-input` | DatePicker (shadcn) | "Fecha de fin" | Debe ser posterior a inicio |
| `periodo-cta` | Button primary | "Continuar" | POST /periodos → redirect paso 2 |
| `periodo-progress` | Progress bar (custom) | 4 dots | Paso actual: filled blue, resto: outline gray |

## States
- **Default:** Form vacío con prellenado inteligente (código sugerido)
- **Validation error:** Campo en rojo + mensaje debajo ("El código ya existe")
- **Success:** Animación breve → redirige automáticamente a paso 2

## API
`POST /api/periodos` → `{ codigo, nombre, fechaInicio, fechaFin, estado: BORRADOR }`
