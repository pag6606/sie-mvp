## Overview

**Page Purpose:** Confirmar que el período anterior está cerrado y ofrecer un CTA claro para iniciar la configuración del nuevo período.

**User Situation:** Alma (coordinadora académica, 38) acaba de autenticarse. Viene de un período anterior cerrado. Quiere ver que todo está en orden y empezar a configurar 2026-2 sin perderse.

**Success Criteria:** Alma identifica en < 3 segundos que 2026-1 está cerrado y hace clic en "Configurar nuevo período" en el primer intento, sin dudar.

**Entry Points:**
- POST /auth/login → redirect según rol (Admin → este dashboard)
- Sesión activa → carga directa

**Exit Points:**
- CTA "Configurar nuevo período" → Crear Período (1.2)
- Quick link "Ver secciones" → Secciones (1.4)
- Quick link "Gestionar usuarios" → Gestión Usuarios (5.1)

## Page States

| State | When | Appearance | Actions |
|-------|------|------------|---------|
| **Default (período cerrado)** | Admin autenticado, período anterior CERRADO | Status card verde + hero card con CTA activo | CTA "Configurar nuevo período" |
| **Período abierto** | Hay un período ABIERTO o EN_CURSO | Status card azul: "2026-2 EN_CURSO" + CTA cambia a "Gestionar período" | CTA → Secciones del período activo |
| **Sin períodos** | Colegio nuevo, nunca configuró período | Status card oculto. Hero card: "Configura tu primer período" | CTA "Crear primer período" |
| **Loading** | Carga inicial de datos del dashboard | Skeleton cards (2 rectángulos grises animados) | Ninguna (espera) |
| **Error** | Fallo al obtener datos del período | Status card roja: "No se pudo cargar la información del período" | Botón "Reintentar" |

## Technical Notes

- **Desktop-first:** El dashboard está optimizado para monitores (1024px+). En tablet se apila verticalmente preservando la jerarquía.
- **Sin menú lateral en MVP:** Los quick links + CTA cubren la navegación. En fase 2 se añade sidebar.
- **Carga optimista:** Si el usuario vuelve al dashboard, usar datos cacheados del estado del período y refrescar en background.
- **A11y:** El CTA principal debe ser focusable y activable por teclado (Enter/Space). Contraste AA en texto sobre fondos de color.
